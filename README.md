# AULAGIT
## Iniciando projeto git 
- Repositório Local
- Repositório Remoto
- git config  --global user.name 'nome'
- git config  --global user.email 'email'
- git init 
- git remote add origin (link do repositório)
## Fazer projeto 
- Faz umas coisas ai
- ## Stage e Commit
- git add * | git add .
- git commit -m 'mensagem do commit'
## Push
- git push origin main | master
